# Cerberus Security Suite
**Versione:** 2.0.0 (Python Edition)

Questa è una suite di sicurezza open-source progettata per l'analisi di sistema e la protezione della rete.

## Funzionalità
- [x] Scansione rapida del sistema (Simulazione)
- [x] Verifica connettività e sicurezza rete
- [ ] Analisi avanzata malware (In sviluppo)

## Come usare
1. Scarica i file.
2. Assicurati di avere Python installato.
3. Avvia `start.bat`.

## Requisiti
- Python 3.x
- Windows 10/11
